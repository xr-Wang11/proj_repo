# Changelog

本项目采用语义化版本的记录方式。

## [0.2.0] - 2026-04-18

### Added

- 新增 `web-vue/`，用 Vue 3 + Vite 搭建并行前端，不改动现有核心计算层
- 新增核心适配层与组合式状态封装，复用现有 `core/`、输入协议和用户函数持久化逻辑
- 新增 Vue 版表达式输入区、结果区、错误区、符号面板、用户函数面板和 JSON 导入导出面板

### Changed

- 修正 Vue 前端中的中文界面文案、按钮说明和交互提示
- 按钮悬停说明补充了 `polar(r, theta, unit)` 中 `unit` 的含义
- Vue 符号按钮增加 `mousedown.prevent`，避免点击后光标被浏览器默认焦点行为打断

## [0.1.0] - 2026-04-18

### Added

- 完成核心计算引擎，包括词法分析、语法分析、复数运算、相量与直角坐标转换
- 支持表达式与结果的 LaTeX 字符串生成
- 提供 CLI 交互入口，支持显示模式、角度单位和用户函数相关命令
- 提供原生网页界面，支持表达式输入、按钮插入、结果显示和错误提示
- 接入用户函数保存、删除、导入导出与持久化
- 提供浏览器单文件构建产物与普通 bundle 构建方式
- 增加按钮悬停说明与常用数学函数入口

### Changed

- 网页输入协议改为支持键盘直接输入常见计算符号
- 增加全角/半角符号归一化处理
- 对数函数从 `log` 调整为 `lg` 与 `ln`
- 网页显示模式、角度单位和部分按钮文案调整为中文

### Notes

- `index.standalone.html` 仅在明确要求时更新
- 当前版本仍处于开发阶段，但已具备基础可用性
